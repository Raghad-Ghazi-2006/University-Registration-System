# University Registration System
 
A Java OOP project that simulates a university registration system for managing students, subjects, and academic records.
 
## Features
- Manage student information
- Register academic subjects
- Sort students by age
- Find successful students
- Find common subjects enrolled in by all students
- Find students who achieved a full mark in at least one subject
 
## Implemented Methods
- Return all successful students from an array of Student objects
- Sort 3 students by age
- Find all common subjects enrolled in by all students
- Return IDs of students who have at least one subject with a full mark
 
## OOP Concepts Used
- Classes and Objects
- Encapsulation
- Arrays of Objects
- Methods
- Object-Oriented Programming
 
## Classes
- Student Class
- Subject Class
- Main Class
